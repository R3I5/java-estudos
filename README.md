# ☕ Java Estudos
Repositório dedicado ao estudo  da linguagem **Java** e dos fundamentos de **Programação Orientada a Objetos (POO)**. Este projeto serve como um guia de referência pessoal, contendo traduções da documentação oficial, resumos teóricos e exercícios práticos.

---

## Estrutura do Repositório

O projeto está organizado para separar a teoria da prática de forma clara e escalável:

* **`resumes/`**: Documentação técnica traduzida e comentada. O foco aqui é entender o "porquê" das coisas, desde conceitos de POO até as nuances da JVM.
* **`src/`**: Implementações práticas e algoritmos. Cada subpasta corresponde a um tópico estudado nos resumos.

---

## Tecnologias e Ferramentas

* **Linguagem:** Java 21 (LTS)
* **IDE:** IntelliJ IDEA
* **OS:** Ubuntu Linux
* **Versionamento:** Git & GitHub
