# Classes
## O que é uma classe? 
### Na sua aplicação, muitas vezes você vai encontrar vários objetos individuais que são semelhantes. Talvez existam milhares de bicicletas, todas da mesma marca e modelo. Cada bicicleta foi construída através dos mesmos diagramas e, além disso, compartilham os mesmos componentes. Nos termos da orientação a objetos, nós falamos que a sua bicicleta é uma instância da classe de objetos conhecidos como bicicleta. A classe é o diagrama do qual objetos individuais são criados.
## A classe Bicicleta a seguir é uma possível implementação de uma bicicleta
```java
    class Bicycle {
    
        int cadence = 0;
        int speed = 0;
        int gear = 1;
        
        void changeCadence(int newValue) {
            cadence = newValue;
        }
        
        void changeGear(int newValue) {
            gear = newValue;
        }
        
        void speedUp(int increment) {
            speed = speed + increment;
        }
        
        void speedDown(int decrement) {
            speed = speed - decrement;
        }
        
        void printStates() {
            IO.println("cadence:" +
            cadence  + " speed:" + 
            speed  + " gear:" + gear);
        }
    }
